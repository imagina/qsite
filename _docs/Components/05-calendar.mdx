# Calendar
Use the [`q-calendar`](https://quasarframework.github.io/quasar-ui-qcalendar/docs) Quasar component to render a calendar with events.

<TableTabs tabsContent={componentCalendar} />

## Usage
Import the `calendar` component and define it within your component:

```jsx
import calendar from '@qsite/_components/master/calendar'

<!--Calendar-->
<calendar :events-data="eventsData" :event-actions="eventActions"/>

//Event Data
eventsData = [
  {
    id: this.$uid(),
    title: 'Event Title...',
    color: '#e57373',
    icon: 'fas fa-calendar',
    image: 'path/file/image',
    date: '2020-01-01 16:00:00',
    mainDetails: [
      { title: 'Title...', icon: 'fas fa-calendar-check', value: 'Description...' },
      { title: 'Title...', icon: 'fas fa-calendar-check', value: 'Description...' }
    ],
    extraDetails: [
      { title: 'Title...', icon: 'fas fa-calendar-check', value: 'Description...' },
      { title: 'Title...', icon: 'fas fa-calendar-check', value: 'Description...' },
      { title: 'Title...', icon: 'fas fa-calendar-check', value: 'Description...' }
    ]
  }
]

//Event Actions
eventActions: [
  {
    label: 'Action label...',
    icon: 'fas fa-pen',
    color: 'green',
    callBack: (event) => {
      // Do anything...
    }
  }
]
```
